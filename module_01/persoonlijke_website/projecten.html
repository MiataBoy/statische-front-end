Mijn projecten

Ik ben in het verleden al actief bezig geweest met nieuwe projecten, en het realizeren ervan. Deze kan je hier terugvinden.

PsychOps Studios

PsychOps was begonnen met het plan om gratis, open-source products te maken. De main focus bij onze producten was altijd om gebruikers iets nieuws en gratis te geven zonder een addertje onder het gras, terwijl ze toch iets krijgen wat ze fijn vinden. PsychOps Studios steunde FOSS altijd, en heeft zich hier ook altijd aan gehouden. Ondanks dit, is PsychOps Studios gestopt omdat deze missie niet meer mogelijk was in de huidige situatie.

Fyuki Games

Fyuki Games is een nederlandse organisatie die werkt aan het maken van een variatie aan projecten. Dit is text-based games, discord bots, websites en meer. Fyuki Games is volledig bezet met nederlandse developers, maar werkt in het engels. Ook Fyuki Games werkt met het FOSS-principe, en zal altijd de code van projecten op Github openbaar zetten. Website & Logo zijn work in progress.
